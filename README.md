## 概述

Solana 区块链监控与自动化工具集，包含链上交易监控和钱包密钥对生成两大核心功能。

## 功能模块

- **链上监控** (`solana_monitor_main.py`) -- 实时监控 Solana 链上交易活动
- **钱包生成** (`solana_wallet_generator.py`) -- 批量生成 Solana 钱包密钥对
